# 任务卡目录索引

> 新 agent 进来不知道先做哪张卡？看这里。本文件是任务卡的**导航索引**，列依赖链和当前状态。
> 单张任务卡细节看对应的 `.md` 文件；全量看板看 AGENTS.md §4。

## 当前推荐入口

- **如果从零接手**：从第一个 ⬜ 的任务开始，往前读它的依赖任务的卡。
- **如果接上一个 agent 的棒**：先读 STATUS.md（增量 handoff），再读 AGENTS.md §3（累计快照），最后定位到当前任务的卡。
